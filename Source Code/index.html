How Can I Enter The IT Industry?

This Open Learning Platform provides you multiple learning paths for different careers in IT. All courses are fully online and self-paced to access at your convenience. They consist of practical exercises curated by the industry and academia to develop the required skills.

How Can I Start My Project Management Career?

The "Project Management Skills" Programme equips you with a broad knowledge base to begin your career in Project Management. All courses are fully online and self-paced to access at your convenience. The programme is aligned with the Certified Associate in Project Management (CAPM) Exam Content Outline of the Project Management Institute, USA.

Career Paths Skills and Requirements
Full Stack Developer Proficiency in HTML, CSS, JavaScript, and knowledge of server-side technologies.
Python Programmer Strong Python coding skills and knowledge of Python frameworks and libraries.
Web Developer Strong Python coding skills and knowledge of Python frameworks and libraries.
Data Scientist Strong analytical skills, proficiency in data analysis tools like Python or R.
Software Tester Attention to detail, knowledge of testing methodologies and tools.
Software Test Automation Engineer Automation skills, scripting languages, familiarity with testing frameworks.
DevOps Professional Knowledge of CI/CD, containerization, cloud services, and automation tools.
Mobile Developer Proficiency in mobile app development languages (e.g., Java, Swift, Kotlin).
UX Professional User research, UI/UX design, usability testing, and wireframing skills.
Security Professional Cybersecurity knowledge, understanding of security tools and best practices.
Cloud Developer Cloud platform expertise (AWS, Azure, GCP), containerization, and scripting skills.
